API Endpoints

Path Method Description Tags
/v1 GET Home Page home
/v1/ad-to-et/{date} GET Get Ethiopian date string from Gregorian date dateConversion
/v1/et-to-ad/{date} GET Get Gregorian date string from Ethiopian date dateConversion
/v1/bahire-hasab/{year} GET Get year's fasting and festival date bahireHasab

Note: The date path variable is in the format 'yy-mm-dd', like 2024-08-12.
